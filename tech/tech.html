What is the difference between Margin, Border, and Padding?

Learning CSS can be confusing and overwhelming at first especially if you are a beginner. Let's talk about Margins, Borders, and Paddings. Let's talk about margins. The image below shows that it has 10 pixels of margin.


This means that 10 pixels of space will be present between the content element and adjacent page elements. If we put multiple elements together, the margin will create a white space between the element to give room.


Now let's talk about padding. Padding is found inside the border of an element. The element shown below uses a padding of 10 pixels on the left and right sides while it uses 15 pixels on both top and bottom sides.


To create the gap, the padding grows the element's size or it shrinks the contents inside.
